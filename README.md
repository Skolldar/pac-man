# Pac-Man Game

A browser based implementation of the classic Pac-Man game using JavaScript, HTML, and CSS.
[![Screenshot-from-2026-02-11-12-40-09.png](https://i.postimg.cc/Y9238CFy/Screenshot-from-2026-02-11-12-40-09.png)](https://postimg.cc/1gTw3PJp)

## Features
- Playable Pac-Man with keyboard controls
- Ghosts with basic AI movement
- Collect cherries and other items
- Score tracking
- Sound effects and images
- Responsive game board

## Project Structure
- `index.html` — Main HTML file
- `index.js` — Entry point for game logic
- `Pacman.js` — Pac-Man character logic
- `Ghost.js` — Ghost character logic
- `ghostmove.js` — Ghost movement algorithms
- `GameBoard.js` — Game board setup and rendering
- `Cherry.js` — Cherry and item logic
- `setup.js` — Game initialization
- `pacman.css` — Stylesheet
- `img/` — Game images and sprites
- `sounds/` — Sound effects

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Use arrow keys to control Pac-Man.

## Requirements
- Modern web browser (Chrome, Firefox, Edge, etc.)
- No installation required
